# Авторизация и управление пользователем

В приложении используется **Firebase Authentication** для
безопасной работы с учетными записями пользователей.

---

## Поддерживаемые функции

- Регистрация по email и паролю
- Вход в систему
- Выход из системы
- Восстановление пароля через email

---

## Процесс авторизации

1. Пользователь вводит email и пароль
2. Данные отправляются в Firebase Authentication
3. При успешной проверке:
   - создается пользовательский документ в Firestore;
   - пользователь перенаправляется на главный экран
4. При ошибке авторизации:
   - пользователь остается на экране входа;
   - отображается сообщение об ошибке

---

## Безопасность

- Все пользовательские данные привязаны к уникальному UID
- Доступ к данным ограничен правилами Firestore
- Пароли не хранятся в открытом виде
