# Избранные блюда

Функция «Избранное» позволяет пользователю сохранять понравившиеся блюда
для быстрого доступа.

---

## Добавление в избранное

- Пользователь может добавить блюдо в избранное из карточки блюда
- Блюдо сохраняется в профиле пользователя (Firestore)

---

## Просмотр избранных блюд

- Все избранные блюда отображаются в отдельной вкладке
- Пользователь может перейти к карточке любого блюда

---

## Хранение данных

Для каждого пользователя сохраняется список избранных блюд:
- mealId
- название
- категория
- изображение
