# Блюдо дня

Функция «Блюдо дня» предназначена для повышения вовлеченности пользователя.

---

## Принцип работы

- Один раз в сутки система выбирает случайное блюдо
- Выбранное блюдо сохраняется для текущей даты
- В течение дня блюдо не меняется

---

## Возможности пользователя

- Просмотр блюда дня
- Переход к карточке блюда
- Добавление блюда в избранное
